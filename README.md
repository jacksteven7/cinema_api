# README

## Settings

* Clone repository 
  git clone https://github.com/jacksteven7/cinema_api.git

* Go inside the project
`cd cinema_api`

* Bundle the project
`bundle install`

* Create databases
`rake db:create`

* Run migrations
`rake db:migrate`

* Seed the database
`rake db:seed`

* Start server
`rails s`
